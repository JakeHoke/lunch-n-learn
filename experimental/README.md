# Experimental Playground

Prototypes exploring interactions impossible (or impractical) in PowerPoint. **Not part of the final deck** — a sandbox to steal ideas from later.

## Access

| Key | Action |
|-----|--------|
| `E` | Open experiment gallery |
| `←` `→` | Navigate between experiments |
| `Esc` | Return to presentation |

## Experiments

1. **Interactive Memory Map** — 3D node graph (hover, click zoom, drag)
2. **Life Timeline** — vertical scroll documentary
3. **Curiosity Engine** — "I wonder..." click chains
4. **Butterfly Effect** — cascade nodes from one click
5. **Living Desk** — clickable objects
6. **The Dot Connector** — Spacebar connects dots
7. **Opportunity Radar** — radar sweep animation
8. **Perspective Shift** — 17 vs 21 split screen
9. **Interactive World** — click countries on globe
10. **The Everything Machine** — spinning gear machine

Each experiment lives in `experiments/NN-name.js` with a `mount(container)` API. No external dependencies.
